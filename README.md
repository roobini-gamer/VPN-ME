# Requirements:
- OpenVpn
- python2

# Instalation:

```
$ git clone https://github.com/roobini-gamer/VPN-ME.git
$ cd VPN-ME
$ sudo mv autovpn2.py /usr/local/bin/autovpn2
```
# Usage:

```
$ sudo VPN-ME <country code>
```
example:``` sudo VPN-ME US ```
